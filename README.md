# 👋 Hello, Welcome to Surfers Common

이 레포지토리는 재활용 가능한 공통 패키지들을 다룹니다.

## Packages

### `@coldsurfers/analytics`

Shared analytics tools for COLDSURF services.

- Registry: GitHub Packages (`https://npm.pkg.github.com`)
- Package: `@coldsurfers/analytics`

Install example:

```bash
yarn add @coldsurfers/analytics react zod
```
